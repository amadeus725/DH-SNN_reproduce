## ⚙️ 技术机制对比

### 神经元动力学

#### Vanilla SNN
```python
# 标准LIF神经元
u[t+1] = β * u[t] + (1-β) * I[t]
if spike: u = V_reset  # 记忆丢失
```

#### DH-SNN
```python
# 多分支树突神经元
for i in range(branches):
    d_i[t+1] = βᵢ * d_i[t] + (1-βᵢ) * I_i[t]  # 持久记忆
u_soma = Σ d_i[t]  # 胞体整合
```

### 关键差异

| 特性 | Vanilla SNN | DH-SNN | 影响 |
|------|-------------|---------|------|
| **重置机制** | 全局重置 | 选择性重置 | 🟢 保持历史信息 |
| **时间尺度** | 单一尺度 | 多时间尺度 | 🟢 并行处理能力 |
| **分支数量** | 1 | 2-8个 | 🟢 功能专业化 |
| **学习能力** | 有限 | 自适应强 | 🟢 任务适应性 |
