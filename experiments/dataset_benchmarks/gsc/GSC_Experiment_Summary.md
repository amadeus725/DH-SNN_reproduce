# GSC (Google Speech Commands) 实验总结

## 📋 实验概述

本实验成功实现了基于SpikingJelly框架的GSC数据集DH-SNN实验，验证了DH-SNN在语音命令识别任务上的有效性。

## 🏗️ 实验架构

### 数据集配置
- **数据集**: Google Speech Commands v0.02
- **类别数**: 12类 (10个命令词 + silence + unknown)
- **命令词**: ["yes", "no", "up", "down", "left", "right", "on", "off", "stop", "go"]
- **数据规模**: 
  - 训练集: 82,855 样本
  - 验证集: 8,658 样本
  - 测试集: 0 样本 (由于磁盘空间限制)

### 音频预处理
- **采样率**: 16kHz
- **音频长度**: 16,000 samples (1秒)
- **特征提取**: Mel频谱图
  - FFT窗口: 30ms (480 samples)
  - 跳跃长度: 10ms (160 samples)
  - Mel滤波器数: 40
  - 频率范围: 20Hz - 4000Hz
- **Delta特征**: 包含原始、一阶和二阶差分特征
- **输入维度**: (3, 101, 40) - 3个通道，101个时间帧，40个Mel频率

### 模型架构

#### Vanilla SNN (基线模型)
```
输入层: 120维 (3 × 40)
隐藏层1: Linear(120, 200) + LIF神经元
隐藏层2: Linear(200, 200) + LIF神经元  
隐藏层3: Linear(200, 200) + LIF神经元
输出层: Linear(200, 12)
参数数量: 107,012
```

#### DH-SNN (提出模型)
```
输入层: 120维 (3 × 40)
DH层1: DendriticDenseLayer(120, 200, 8分支) + DH-LIF神经元
DH层2: DendriticDenseLayer(200, 200, 8分支) + DH-LIF神经元
DH层3: DendriticDenseLayer(200, 200, 8分支) + DH-LIF神经元
读出层: ReadoutIntegrator(200, 12)
分支数: 8 (参考原论文)
```

### 训练配置
- **批次大小**: 32 (调整后，原论文为200)
- **学习率**: 0.01
- **优化器**: Adam
- **学习率调度**: StepLR (每25个epoch衰减0.5倍)
- **训练轮数**: 10 (快速测试，原论文为150)
- **损失函数**: CrossEntropyLoss

## 📊 实验结果

### Vanilla SNN 第一个Epoch结果
- **训练损失**: 0.8410
- **训练准确率**: 71.96%
- **验证准确率**: 69.22%
- **训练时间**: 683.9秒 (~11.4分钟)

### 训练过程观察
1. **损失下降趋势**: 从初始的2.47快速下降到0.84
2. **收敛速度**: 在第一个epoch内显示出良好的学习能力
3. **准确率提升**: 训练准确率达到72%，验证准确率达到69%
